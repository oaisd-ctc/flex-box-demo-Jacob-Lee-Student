Flex-Box Examples

Flex-Box Container Properties

flex-direction

The flex-direction property effects which direction the container stacks items

flex-direction: row; (default)

flex-direction: column;

flex-wrap

The flex-wrap property controls whether or not the flex items in the container should wrap
Resize the page to see how the flex-wrap property effects the content

flex-wrap: nowrap; (default)

flex-wrap: wrap;

flex-flow

The flex-flow property is a combination of the flex-direction and the flex-flow properties

flex-flow: row wrap;

Flex-Box Item Properties

order

The order property controls the order that flex items will appear

1 order: 3;

2 order: 1;

3 order: 4;

4 order: 2;

flex-grow

The flex-grow property controls how flex items will grow relative to the other flex items

flex-grow: 1;

flex-grow: 2;

N/A

flex-grow: 3;

align-self

The align-self property controls the alignment of an item

N/A

align-self: center;

align-self: flex-start;

align-self: flex-end;

Extra Credit

:hover

The :hover psuedo-class specifies that when an element is being hovered over by the mouse, an action is preformed.
This can be changing the color, size, or visability of an element

 

 

 

Using the flex-grow property in our :hover psuedo-class we can expand one of our div.
If we add a flex property with a value of 1 in our div declaration it'll control the div's basis and how much it'll shrink when a different div is hovered over.

Example Code

<!DOCTYPE html>
<html>
    <head>
        <style>
            #container {
                display: flex; /*creates a flex container*/
                margin-left: auto;
                margin-right: auto;
                width: 500px;
            }
            
            #container div {
                background-color: black; 
                border: 2px solid white;
                border-radius: 10px;
                flex: 1; /*controls the basis and the shrink value*/
                transition-duration: 1s; /*tells the browser how long the animate will play for*/
            }

            #container div:hover { /*when a div is hovered over, it will apply these changes to it*/
                background-color: white;
                color: black; 
                flex-grow: 2; /*tells the browser to expand the div that is being hovered over*/
            }
        </style>
    </head>
    <body>
        <div id="container"> <!--this is the html to make the above container work-->
            <div><p>&nbsp;</p></div>
            <div><p>&nbsp;</p></div>
            <div><p>&nbsp;</p></div>
        </div>
    </body>
</html>
